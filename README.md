Proiect: Manager Statistică Jucători

    Gestionare Jucători

    Adăugare: Bagi un jucător nou în listă.

    Ștergere: Scoți un jucător din listă.

    Căutare: Găsești rapid un jucător după ID.

    Listă Completă: Vezi tabelul cu toți jucătorii tăi.

    Statistici a Jucătorului

    Damage Dealt: Cât de mult a lovit.

    Healing: Cât de mult a vindecat.

    Damage Taken: Cât de multă bătaie a încasat.

    Personaj: Pe ce erou a jucat.

    Roluri

    Categorii: Împarte jucătorii pe categorii: Tank, Support sau DPS.

    Sortare: Poți să vezi doar jucătorii de tip "Support", "DPS" sau "Tank".

    Calcul de Performanță

    Analiză: Aplicația îți spune automat dacă un jucător este Bun sau Slab, bazându-se pe cifrele lui (Damage/Healing/Damage Taken) =======
